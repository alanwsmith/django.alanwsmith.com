  3. Install the notes app in our Django project by updating INSTALLED_APPS in the tutorial_site/tutorial_site/settings.py file to add :

    update for: tutorial_site/notes/models.py
    from django.db import models

class Note(models.Model):
    note_text = models.CharField(max_length=200)
    created_at = models.DateTimeField("Date")

    def __str__(self):
        return self.note_text
