Section Steps

  1. Add the url path route

    python
    path("<int:note_id>/edit/", views.edit, name="edit"),

  2. This makes an intentional terminal error : 

    AttributeError TKTKTKTK

  3. Add "edit" to the view :

    python
    @login_required
def edit(request, note_id):
    note = get_object_or_404(Note, pk=note_id)
    return render(request, "notes/edit.html", {"note": note})

  4. That clears the error

  5. Add the link to the edit page :

    html
    <div>
    <a href="{% url 'notes:edit' note.id %}">edit</a>
</div>
