Section Steps

  1. Add to notes urls.py

    python
    path("<int:note_id>/", views.detail, name="detail"),

  2. There's a terminal error here

  3. Update the notes index page to have links to the individual notes

    html
    <a href="{% url 'notes:detail' note.id %}">view note</a>

  4. Produces a terminal error (this might be a duplicate)

    AttributeError : module 'notes.views' has no attribute 'detail'

  5. Add to notes/views.py

    python
    from django.shortcuts import get_object_or_404

    and

    python
    @login_required
def detail(request, note_id):
    note = get_object_or_404(Note, pk=note_id)
    return render(request, "notes/detail.html", {"note": note})

  6. This clears the error

    Visit the /notes page and you'll see the links. Click one and you'll get a TemplateDoesNotExist error. for "notes/detail.html"

  7. Create the notes/details.html template

    html
    <!DOCTYPE html>
<html>
<body>
  <h1>Note Detail</h1>
</body>
</html>

  8. Now show the details for the note

    html
    <!DOCTYPE html>
<html>
<body>
  <h1>Note Detail</h1>
  <div>Text: {{ note.note_text }}</div>
  <div>Date: {{ note.created_at }}</div>
</body>
</html>

  9. Add links back to the notes index page

    html
    <!DOCTYPE html>
<html>
<body>
  <div>
      <a href="{% url 'notes:index' %}">Notes</a>
  </div>
  <h1>Note Detail</h1>
  <div>Text: {{ note.note_text }}</div>
  <div>Date: {{ note.created_at }}</div>
</body>
</html>
