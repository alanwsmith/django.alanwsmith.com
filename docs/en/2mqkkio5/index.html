The Files

  1. Make the templates direcotry

    command line: 
    mkdir -p templates/registration

  2. Add the includes import and auth include to site_files/urls.py. The file should look like this:

    update for: site_files/urls.py
    from django.contrib import admin
from django.urls import path
from django.urls import include

urlpatterns = [
    path("accounts/", include("django.contrib.auth.urls")),
    path("admin/", admin.site.urls),
]

  3. Add login/logout default redirects to the bottom of the site_files/settings.py file

    add to: site_files/settings.py
    LOGIN_REDIRECT_URL = "/"
LOGOUT_REDIRECT_URL = "/"

  4. Add the login file:

    new file: templates/registration/login.html
    <!DOCTYPE html>
<html>
<body>
    <form method="post">
      {% csrf_token %}
      {{ form }}
      <input type="submit" value="Log In" />
    </form>
  </body>
</html>
