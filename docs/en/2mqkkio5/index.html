Add Basic User Authencation

Introduction

  • These are my notes for how to set up basic user auth

  • The notes assume you've already followed the steps in How-To: Create A New Django Site and Add Basic Template And Static File Handling In Django

  • Right now it only sets up so that login/logout can be used with the built-in users in a dev environment

  • The notes don't have a lot of details with them right now. It's mainly the code lines themselves. I'll add more detail over time.

  • TBD on what needs to change before deploying to production

The Steps

  1. Add the includes import and auth include to site_files/urls.py. The file should look like this:

    update for: site_files/urls.py
    from django.contrib import admin
from django.urls import path
from django.urls import include

urlpatterns = [
    path("accounts/", include("django.contrib.auth.urls")),
    path("admin/", admin.site.urls),
]

    The specific lines that were added are:

    url import includes line
    from django.urls import include

    And urlpatters = []:

    path include line
    path("accounts/", include("django.contrib.auth.urls")),

  2. Add login/logout default redirects to the bottom of the site_files/settings.py file

    add to: site_files/settings.py
    LOGIN_REDIRECT_URL = "/"
LOGOUT_REDIRECT_URL = "/"

  3. Make the templates direcotry

    command line: 
    mkdir -p templates/registration

  4. Add the login file:

    new file: templates/registration/login.html
    <!DOCTYPE html>
<html>
<body>
    <form method="post">
      {% csrf_token %}
      {{ form }}
      <input type="submit" value="Log In" />
    </form>
  </body>
</html>
